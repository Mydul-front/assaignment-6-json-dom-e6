Frequently Asked Questions

1.What is the difference between var, let and const?
var -> Function scoped, can be reassigned, hoisted with undefined (avoid using it). let -> Block-scoped, can be reassigned, hoisted but in Temporal Dead Zone (TDZ). const -> Block scoped, cannot be reassigned, hoisted but in TDZ.
2. What is the difference between map(), forEach(), and filter()
map() -> Returns a new array by transforming each element. forEach() -> Iterates over elements but does not return anything. filter() -> Returns a new array with elements that match a condition.
3. What are arrow functions, and how are they different from regular functions?
Arrow functions (=>)-> Shorter syntax, no own this, no arguments, cannot be used as constructors. Regular functions (function)-> Have their own this, support arguments, can be used as constructors.
4. How do JavaScript Promises work?
A Promises in JavaScript represents a future async operation with three states: pending, fulfilled and rejected.
5. How does closure work in Javascript?
A closure in JavaScript is when a function remembers variables from its outer scope even after the outer function has finished executing.
